# Changelog

All notable changes to WordLens are documented in this file.

## [1.2.0] - 2026-06

### Fixed
- Icon and popup now anchor correctly next to the selected text (previously could drift to the top of the page or appear below the selection instead of above it)

### Added
- Voice picker — choose from up to 4 real system voices detected via the Web Speech API, persisted across sessions
- Inline "Know more ▾" section under the translation — expands in place to show word origin and usage context, without opening a new tab

## [1.1] - 2026-06

### Added
- Audio pronunciation (🔊) with selectable voice and adjustable speed
- Translation history (last 50 lookups, stored locally)
- Language dropdown — 15 selectable target languages (previously fixed to Hindi)
- Right-click context menu option to translate without the floating icon
- Toolbar settings panel with On/Off toggle, Settings/History/More tabs

## [1.0] - 2026-06

### Added
- Initial release
- Text-selection floating icon and in-page popup
- Translation via MyMemory API
- Definitions, synonyms, and word origin via Free Dictionary API
- "Deep Dive" external links to Google, Wiktionary, and Etymonline
